Virtual Healthcare Hub (A group project)

About my Project(this project is not hosted)

This project was developed with a paitent first mindset , aiming to reduce the time paitents spent waiting in hospital queues. This platform allows users to book appointments with doctors from specific department , minimizing physical waiting times . It also includes the option for scheduled video consultations , enhancing remote accessibility . The website offers informative content about leading health insurance providers and includes tools to calculate BMI , check obesity status , and understand basic blood pressure level.

Built using HTML, CSS, Java script and BootStrap, the project emphasizes clean navigation through interlinked pages such as Home, About , Appointment Form and Contact. It also focuses on structured content presentation using tables and visual elements.
